#   • Windows uses the bundled CEF (Chromium) backend.

#     below keeps it off Linux/Raspberry Pi entirely.
#
#     On Linux you also need the system WebView libraries, e.g. on Debian/Ubuntu:
#       sudo apt install gir1.2-webkit2-4.1 python3-gi gir1.2-gtk-3.0
#     (or `pip install pywebview[qt]` plus PyQt/PySide if you prefer the Qt backend).
