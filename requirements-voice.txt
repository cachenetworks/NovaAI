# ─────────────────────────────────────────────────────────────────────────────
# NovaAI — VOICE / ML extras (optional).
#
# Install on top of the base requirements when you want speech input, spoken
# replies, local memory embeddings, or singing:
#   pip install -r requirements.txt -r requirements-voice.txt
#
# Heads-up for Raspberry Pi / ARM: torch + coqui-tts are large and XTTS voice
# synthesis runs on CPU there, so it will be slow. You only need these if the
# Pi actually has a microphone/speakers. For a headless text/web box, skip this.
#
# GPU users: after installing, optionally replace torch/torchaudio with a CUDA
# build, e.g.:
#   pip install --upgrade --index-url https://download.pytorch.org/whl/cu128 torch torchaudio
# ─────────────────────────────────────────────────────────────────────────────
